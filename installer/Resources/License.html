Taufinity CLI — Privacy Notice

By clicking Agree and installing the Taufinity CLI, you accept the following data collection practices.

What we collect

  • A random device identifier (UUID) — not linked to any personal information
  • CLI version, operating system, and processor architecture
  • Which commands you run (not their arguments or any file contents)
  • Update check and install / uninstall events

What we do not collect

  • Your name, email address, or any personal information
  • Command arguments, file paths, or file contents
  • Passwords, tokens, or credentials of any kind

Storage and retention

Data is stored in the EU (Netherlands, europe-west4) and retained for 90 days.

Opt out

Set TAUFINITY_NO_TELEMETRY=1 in your shell environment to disable all data collection before or after installation.
